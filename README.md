## Arquivos
- `index.html`: conteúdo do site
- `style.css`: aparência
- `script.js`: contagem regressiva e menu

## Antes de publicar
No arquivo `index.html`, procure por:
1. `Em breve, insira aqui` — substitua pelo nome e endereço do local.
2. `href="#"` no botão "Como chegar" — substitua pelo link do Google Maps.
3. Se desejar, adicione depois as informações de PIX.

O link de confirmação já está configurado:
https://forms.office.com/r/406JzaxBcX

## Publicar no GitHub Pages
1. Crie um repositório público no GitHub, por exemplo: `casamento-tatiane-willian`.
2. Envie `index.html`, `style.css` e `script.js` para a raiz do repositório.
3. Abra `Settings` > `Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch `main` e a pasta `/ (root)`.
6. Clique em `Save`.
7. Aguarde alguns minutos até o GitHub mostrar o endereço publicado.

## Fotos
Esta primeira versão não usa fotos. Isso facilita a publicação inicial.
Depois, você pode criar uma pasta `images` e incluir fotos do casal no site.
